# 💰 Auditoría de Rentabilidad y Optimización de Márgenes

## 📌 Contexto del Proyecto
En este análisis, utilicé **Python** para evaluar la rentabilidad de un portafolio de productos financieros. Como **Ingeniero Industrial**, mi objetivo fue identificar productos con márgenes ineficientes y proponer ajustes estructurales en los costos para mejorar la salud financiera de la operación.

## 🛠️ Herramientas Utilizadas
* **Lenguaje:** Python (Librería Pandas).
* **Conceptos Aplicados:** Margen de contribución, análisis de costos variables y optimización de precios.
* **Visualización:** Power BI para la creación de Dashboards gerenciales.

## 📊 Hallazgos Clave
* **Identificación de Fugas:** Se detectó que la "Tarjeta Joven" operaba con un margen crítico del **15.5%**, muy por debajo del estándar deseado de rentabilidad.
* **Propuesta de Mejora:** Diseñé un escenario de optimización reduciendo los costos variables, lo que permitió proyectar un incremento del margen al **33.3%**.
* **Eficiencia de Productos:** Clasificación estratégica de activos:
    * **Cuenta de Ahorro:** Alta rentabilidad porcentual (87%).
    * **Crédito Micro:** Mayor generación de masa de margen (flujo de caja nominal de $140.000).

## 🚀 Impacto en el Negocio
Este script y el análisis posterior permiten a la gerencia tomar decisiones basadas en datos para:
1. **Optimizar el Portafolio:** Liquidar o reestructurar productos con baja rentabilidad.
2. **Control de Costos:** Ajustar políticas de costos operativos basadas en el margen real.
3. **Estrategia de Precios:** Definir precios sugeridos que garanticen la sostenibilidad financiera.
